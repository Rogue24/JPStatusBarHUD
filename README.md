# JPStatusBarHUD
简陋的状态栏指示器

## 显示自定义文本
[JPStatusBarHUD showMessage:@"哈哈"];

## 显示自定义图文 
[JPStatusBarHUD showMessage:@"曹操" image:[UIImage imageNamed:@"star"]];

## 显示成功信息 
[JPStatusBarHUD showSuccess:@"加载成功"];

## 显示错误信息 
[JPStatusBarHUD showError:@"加载失败"];

## 显示加载信息 
[JPStatusBarHUD showLoading:@"正在加载中..."];

## 隐藏HUD 
[JPStatusBarHUD hide];
